Quick Dotfiles Setup

Vim

Backup your old Vim configuration and download the new one:

mv ~/.vimrc ~/.vimrc.backup.$(date +%Y%m%d%H%M%S) && curl -fsSL https://raw.githubusercontent.com/LeanderZiehm/dotfiles/main/sync/.vimrc -o ~/.vimrc
Command copied to clipboard!

Neovim

Backup your old Neovim configuration, create config folder if missing, and download the new one:

mv ~/.config/nvim/init.lua ~/.config/nvim/init.lua.backup.$(date +%Y%m%d%H%M%S)
mkdir -p ~/.config/nvim
curl -fsSL https://raw.githubusercontent.com/LeanderZiehm/dotfiles/refs/heads/main/sync/.config/nvim/init.lua -o ~/.config/nvim/init.lua
Command copied to clipboard!

Tmux

Download Tmux configuration:

curl -fsSL https://raw.githubusercontent.com/LeanderZiehm/dotfiles/refs/heads/main/sync/.tmux.conf -o ~/.tmux.conf
